# Introducción a los algoritmos

Los algoritmos son el primer paso hacía la programación y la resolución de problemas usando la lógica, un programador debe desarrollar un pensamiento algoritmico para poder solucionar problemas usando las herramientas que la computación le provee.

Un algoritmo es un conjunto finito y ordenado de instrucciones que aplicadas de manera lógica permiten resolver un problema. Por ejemplo, una receta de un pastel contiene una serie de pasos ordenados para preparar y hornear el pastel, usted debe seguir los pasos en orden para poder llegar al resultado. Pues los algoritmos funconan muy similar a las recetas, con la diferencia que el que lo va a leer o ejecutar será un mecanismo lógico (en este caso la computadora).
