# C cómo primer lenguaje de programación

C es un lenguaje realmente simple con el que se puede adentrar y conocer muy de cerca el funcionamiento real de un lenguaje de programación y la gestión de memoria, C es el lenguaje padre de la informática moderna pues gracias a este lenguaje existe un 70% de los sistemas modernos, siendo el protagonista en muchos sistemas operativos, lenguajes de programación, drivers, software de escritorio, servidores, redes, gráficos, etc.
