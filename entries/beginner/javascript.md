# Javascript como primer lenguaje de programación

El principal enfoque de este lenguaje es en efecto la web, por lo que es recomendable aprender HTML primero para una mejor experiencia con este lenguaje.

Html es demasiado fácil por lo que no te va a tomar más de una hora entenderlo.

- [Tutorial de HTML](https://developer.mozilla.org/es/docs/Learn_web_development/Core/Structuring_content)
- [Curso de HTML (Revisar solo la parte de HTML)](https://www.youtube.com/watch?v=MJkdaVFHrto)

Una vez ya tenemos en cuenta html, lo mejor para aprender javascript es usando el navegador como interprete para despues aprender un entorno diferente como node, deno o bun.

- [Introducción a javascript - Mozilla](https://developer.mozilla.org/es/docs/Web/JavaScript/Guide/Introduction)

Cursos:
- [Curso completo de javascript - FreeCodeCamp](https://www.freecodecamp.org/espanol/news/aprende-javascript-curso-completo-desde-cero/)
- [Javascript completo para principiantes](https://www.youtube.com/watch?v=QoC4RxNIs5M)
